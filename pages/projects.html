Shoot Messenger - 2022

TCP Messaging application built in Java

Status: Server and client users can connect and chat on a local network.
Quirks: Runs in terminal.

  • Users can create or connect to server rom any internet connected computer or virtual machine with a modern OS.
  • Each server is unique with its own list of users, configured settings and chat history.
  • The state of each server maintained in flat files local to the server machine.
